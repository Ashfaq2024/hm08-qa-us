# hm08-qa-us

## Description
This project is designed to automate the process of ordering a taxi through Urban Routes. The application allows users to set their address, select a support plan, and make payments while providing a seamless user experience.

## Technologies Used
- **Node.js**: A JavaScript runtime for building server-side applications.
- **Express**: A web application framework for Node.js that simplifies routing and middleware management.
- **Mongoose**: An ODM (Object Data Modeling) library for MongoDB and Node.js, providing a schema-based solution to model application data.
- **Morgan**: An HTTP request logger middleware for Node.js, useful for logging requests.
